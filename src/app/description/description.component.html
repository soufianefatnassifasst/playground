Un exemple de choses faites dans notre application

L'objectif est d'implémenter un nouvel écran en utilisant qui apparaitra au clic sur le lien "fonctionnalité". Les données sont mises à disposition dans le service HttpMocksService et les types sont décrits dans domain.ts. Il faudra d'abord récuper la liste des id de contrats dans getContrats(), pour chacun d'entre eux, faire un appel à getConsommation(idDuContrat) puis intégrer les différentes valeurs. Nous pretterons attention à :
  • l'efficacité des appels au services HTTP ( temps total de chargement, caches lors d'allers/retours via la navigation )
  • Au découpage composants/services.
